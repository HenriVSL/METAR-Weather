# METAR Weather

## Why?

Goal of this project was to acquire weather information for airport METAR data and display it in a format that I prefered. 

In my daily worklife I need the temperature, windspeed and visibility information to be easily availabe on some of the nearby airports. So I decided to try out how does this "new" AI code development work. This was mostly done by Claude Sonnet 4.6 and Opus 4.8 on medium to high. In the begining I also used Gemini Pro.

I would like to try to have this published to App Store. 

## In practice

User can add airports with ICAO code. This ICAO code is used to fetch airport basic info like name and any frequencies if they list them. Then weather info is acquired. All data is provided by aviationweather.gov Thank you for that!
